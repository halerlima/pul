# PUL — Protocol for Universal Settlement

> **An open protocol for universal value transfer and settlement.**

PUL (Protocol for Universal Settlement) is an open protocol designed to enable the transfer and settlement of value between systems, applications, organizations, and individuals without relying on a specific bank, payment gateway, or centralized provider.

Just as **TCP/IP** became the universal protocol for transferring data across the Internet, **PUL** aims to become an open standard for transferring value.

> **PUL is not a cryptocurrency.**
>
> **PUL is not a bank.**
>
> **PUL is not a payment company.**
>
> **PUL is an open protocol.**

---

# Why PUL?

Today's payment ecosystem depends on multiple intermediaries, proprietary APIs, and country-specific infrastructures.

PUL proposes a different approach:

* Open specification
* Decentralized architecture
* Vendor-neutral
* Secure by design
* Interoperable
* Community-driven

Any developer should be able to build software compatible with the protocol.

---

# Vision
